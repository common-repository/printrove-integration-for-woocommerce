With the Printrove for Woocommerce application, you will be able to push your Woocommerce store orders automatically to Printrove's Merchant Panel. You won't have to place each order manually.

== Description ==
This plugin is used to push orders from your WooCommerce Backend to the Printrove Merchant Panel. Upon installing and activating the plugin, click on Printrove for WooCommerce in the menu.
Login to your Printrove account.
Go to Store Orders and click on WooCommerce > Connect. 
Follow the steps to connect your WooCommerce store to Printrove.

This plugin also adds the following changes in the Checkout Page
1. Address Line 3 is added
2. All address fields are limited to 30 characters
3. Pincode is limited to 6 characters

For more details, you can visit community.printrove.com
For any queries please email us at info@printrove.com or call at +91 44 45522005

== Installation ==

Step 1: Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.

Step 2: Click Install and once the installation is complete, Activate the plugin through the 'Plugins' screen in WordPress.

Step 3: Click on Printrove for Woocommerce in the left menu bar

Step 4: Login to your Printrove account

Step 5: Go to Store Orders and click on WooCommerce > Connect

Step 6: Follow the steps to connect your WooCommerce store to Printrove

== Frequently Asked Questions ==
 - Where do I find the API Key? - 
 Visit merchants.printrove.com and go to Profile > API Key to generate and copy your API key.
